# Weenus Token

Weenus is a test ERC20 token created by BokkyPooBah for development and testing purposes.

## How to Get Test Tokens

Send a 0 value transaction from your account to the token contract address, and the contract will mint 1,000 WEENUS tokens for your account.

## Token Details

- **Decimals:** 18
- **Faucet Amount:** 1,000 tokens per request
